# quantum_version_RAG
This is the hybrid quantum-classical RAG for document chatting(earlier project). A quantum version of my classical RAG AGENT project. Refer to to quantum_version_RAG.docx file for project explaination and methodology. 
## 5. How to Reproduce

### Clone the Repository
```bash
git clone https://github.com/SARPAT/quantum_version_RAG.git
cd quantum_version_RAG
```
## Install dependencies
```bash
pip install pennylane qiskit-ibm-runtime langchain-nvidia-ai-endpoints faiss-cpu
```
